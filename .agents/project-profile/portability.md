# Portability Profile

This profile is intentionally generic. Before copying it into a repository:

1. Replace placeholder source roots in `surfaces.md`.
2. Replace placeholder commands in `validation.md`.
3. Replace placeholder playbook paths in `playbooks.md`.
4. Update `docs-map.md` to match the repository's active docs.
5. Adjust `docs-system.md` if the repository has optional layers or naming exceptions.

Keep project facts in `.agents/project-profile/`; keep reusable logic in `.agents/skills/`.
