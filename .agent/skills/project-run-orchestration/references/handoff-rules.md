# Handoff Rules

Update `handoff.md` whenever stopping a complex run.

## Required Sections

- Current State
- Last Completed Task
- Next Recommended Task
- Blockers
- Validation State
- Docs Sync State
- Scope Notes
- Context For Next Agent

## Quality Bar

A new Agent should be able to continue by reading:

1. `AGENTS.md`
2. `.ai/README.md`
3. the run folder
4. relevant docs named in the run folder

The handoff should not require hidden chat context.
