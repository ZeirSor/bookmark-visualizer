# ADR 0004: 插件元数据保存在本地并支持导入导出

## 状态

已接受

## 背景

浏览器原生书签没有备注和摘要字段。产品需要保存这些补充信息，同时避免引入云端服务和隐私风险。

## 决策

使用 `chrome.storage.local` 保存备注、摘要、UI 偏好和导入导出状态。提供元数据导出和导入能力。

## 替代方案

- 使用 `chrome.storage.sync`：可跨设备同步，但容量较小，不适合大量备注和摘要。
- 使用远程服务：能力更强，但增加账号、隐私和运维成本。
- 不保存备注摘要：无法满足卡片字段需求。

## 后果

- 数据默认留在本机。
- 用户换设备时需要手动导出导入插件元数据。
- 后续需要维护 metadata version 和迁移机制。
