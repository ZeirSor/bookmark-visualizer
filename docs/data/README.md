# Data

本目录保存本地存储、数据模型和导入导出设计。

## Active Docs

- [Storage](storage.md)：当前数据源原则、`chrome.bookmarks` 原生数据、`chrome.storage.local` 元数据和隐私边界。
- [Phase 2 data model and import/export](phase-2-data-model-import-export.md)：未来数据模型、多维表格和导入导出设计指南。

## Maintenance

当 storage key、metadata schema、migration、导入导出格式或长期数据模型发生变化时，更新本目录。若改变“原生书签作为结构事实来源”原则，必须新增或更新 ADR。
