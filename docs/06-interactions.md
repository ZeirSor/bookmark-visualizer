# 交互规则

## 点击

- 点击浏览器工具栏图标：打开 Bookmark Visualizer 完整工作台标签页。
- 点击左侧文件夹：右侧展示该文件夹直接子书签。
- 点击左侧文件夹行：同时选中文件夹并展开 / 折叠。
- 点击左侧书签条目：右侧切换到该书签所在文件夹，自动滚动并高亮对应卡片。
- 点击右侧卡片空白区域：新标签页打开书签 URL。
- 点击右侧卡片右上角打开图标：新标签页打开书签 URL。
- 点击标题或备注编辑入口：进入编辑，不触发打开。

## 搜索

- 输入关键词后进入全局搜索状态。
- 当前搜索匹配标题和 URL；备注和摘要搜索后续实现。
- 清空关键词后回到当前文件夹视图。
- 搜索结果展示卡片和所在路径。
- 过滤器属于后续能力，当前尚未实现。

## 拖拽书签

- 右侧卡片可拖到左侧文件夹。
- drop 前高亮合法目标。
- 拖拽到左侧长文件夹树时，靠近树顶部 / 底部会自动滚动文件夹树。
- 不可修改文件夹不接受 drop。
- 移动成功后刷新树和卡片区。
- 移动成功后显示撤销提示。

## 拖拽文件夹

- 文件夹拖拽属于后续能力，当前尚未实现。
- 根节点、浏览器特殊不可修改节点不可拖动。
- 文件夹不能移动到自身或自身子孙节点。
- 移动文件夹前应显示清晰的目标反馈。

## 右键菜单

- 当前支持右键书签卡片，菜单包含“编辑”“移动”“删除”。
- 悬浮“移动”后展示多级文件夹级联菜单，不使用内部滚动选择框。
- 多级文件夹级联菜单应保留短暂关闭缓冲，允许鼠标跨过父子菜单之间的间隙。
- 选择目标文件夹后调用真实移动。
- 右键文件夹和可搜索文件夹选择器属于后续能力。
- 当前节点所在文件夹可以显示但不可作为无意义移动目标；它的子文件夹仍必须可继续展开并作为目标。

## 新建

- 新建书签和新建文件夹属于后续能力，当前 UI 尚未暴露。
- 新建成功后刷新当前文件夹。
- 新建失败时不保留虚假 UI 项。

## 编辑

- 书签可编辑标题、URL、备注和摘要。
- 当前书签编辑支持标题、URL、备注；摘要和文件夹名称编辑后续实现。
- 标题为空时需要阻止保存。
- URL 必须是浏览器可打开的合法地址。

## 删除

- 删除书签需要确认或提供明显撤销。
- 当前删除书签需要确认，成功后进入操作日志并可撤回。
- 删除文件夹属于后续能力，必须确认，因为会影响子项。
- 删除成功后从 UI 移除。
- 若撤销无法完整恢复原结构，应在提示中说明。

## 摘要抓取

- 用户点击“获取摘要”后触发。
- 若没有目标站点权限，先请求授权。
- 授权后读取页面 description。
- 抓取失败时保留手动填写入口。

## 撤销

- 操作成功后显示 toast。
- 移动撤销恢复原 parentId 和 index。
- 编辑撤销恢复旧字段。
- 删除撤销优先恢复书签；当前通过重新创建书签恢复，因此会生成新的 bookmarkId。
- 文件夹删除撤销如无法完整恢复，需要明确提示限制。
